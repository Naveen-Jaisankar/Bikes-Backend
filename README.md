# DUBLIN Bikes

This Project is the server for Dublin bikes.

## Installation

To install the related directories use command

```bash
pip install -r requirements.txt
```

To set the flask environment use command
```bash
set FLASK_ENV=app
```

To run the application use command
```bash
flask run
```
